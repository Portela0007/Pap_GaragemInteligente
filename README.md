RESUMO
Este projeto teve como objetivo desenvolver uma garagem inteligente controlada remotamente através de uma aplicação móvel, integrando conceitos de automação, eletrónica e Internet das Coisas (IoT). O sistema permite abrir e fechar o portão da garagem à distância, utilizando um ESP com Wi-Fi como microcontrolador principal. Para garantir maior funcionalidade e segurança, foram adicionados um sensor de movimento, responsável por ativar automaticamente a iluminação, e servomotores, que realizam o movimento do portão.
Ao longo do desenvolvimento, foram aplicados conhecimentos de programação, comunicação sem fios, montagem de circuitos, eletrónica e modelação física, através da construção de uma maquete em madeira. A aplicação móvel criada permite o controlo do sistema em tempo real, oferecendo ao utilizador uma experiência prática, segura e eficiente.
Os testes efetuados demonstraram que o sistema apresenta boa fiabilidade, tempo de resposta adequado e funcionamento estável, comprovando a eficácia da automação implementada. O projeto revelou-se uma oportunidade de consolidar aprendizagens adquiridas no curso e de desenvolver competências essenciais para a área das tecnologias e automação.
